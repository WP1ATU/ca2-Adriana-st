About This Application

Car Collection Manager

CA2 – Part 1 | Web Development

This application allows you to manage a personal car collection. You can add cars to your collection, view all entries, and remove any that you no longer wish to track.

Features
  • Add cars by providing make, model, year, and an optional image URL
  • View your full collection with images displayed in a clean list
  • Delete individual entries from the collection
  • Data is stored persistently in a MongoDB database
